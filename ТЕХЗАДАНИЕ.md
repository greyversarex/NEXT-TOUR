## Архитектура
- **Frontend**: React + TypeScript + Vite + TailwindCSS + shadcn/ui + recharts + date-fns
- **Backend**: Express + TypeScript + Drizzle ORM
- **DB**: PostgreSQL (sessions + password_reset_tokens + settings)
- **Auth**: Passport.js (Local Strategy) + express-session
- **File Upload**: multer → client/public/uploads/
- **New DB Tables**: `password_reset_tokens`, `settings`
